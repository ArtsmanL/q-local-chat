## 🚀 快速开始

### 📋 环境要求
- Node.js 14.0+
- 局域网环境

### ⚡ 安装运行

```bash
# 进入项目目录
cd QChat

# 安装依赖
npm install

# 启动服务器
node chat-server.js
```

### 🌐 访问地址
启动成功后会显示：
```
✅ 服务器已启动!
🏠 本机访问: http://localhost:3000
🌐 局域网访问: http://192.168.x.x:3000
```

邀请局域网小伙伴一起玩吧！👋

## 🎮 小游戏介绍

### 🎯 五子棋

> 双人对战，智谋较量！先连成五子者胜！

**🎯 游戏规则**
- ⚫ 黑棋先手，⚪ 白棋后手
- 轮流落子，五子连珠获胜
- 横、竖、斜都算连线

**✨ 游戏特色**
- 15×15 标准棋盘
- 落子动画 + 胜利烟花
- 支持观战模式

---

### 🎣 金钩钓鱼

> 童年经典扑克游戏！钓鱼收牌，其乐无穷！

**🎯 游戏规则**
- 🃏 每人7张手牌开局
- 出牌到鱼塘，相同点数即可"钓鱼"
- 钓走匹配牌及中间所有牌
- 手牌最先出完者获胜！

**💡 小技巧**
- 记住鱼塘有哪些牌
- 策略性出牌钓大鱼
- 2-4人都可玩哦~

---

### 🎴 翻翻乐

> 双人记忆大挑战！翻开配对，谁的记忆力更强？

**🎯 游戏规则**
- 🎴 16张卡牌正面朝下
- 轮流翻开两张卡牌
- ✅ 配对成功 → 得1分，继续翻
- ❌ 配对失败 → 翻回，换对方
- 得分高者获胜！

**✨ 游戏特色**
- 精美卡牌图案
- 翻牌动画特效
- 考验瞬间记忆

## 🎯 游戏流程

### 创建游戏
1. 点击聊天室右侧 🎮 游戏按钮
2. 选择想玩的游戏
3. 点击 "立即组队" 🚀

### 加入游戏
- 聊天室会弹出游戏邀请卡片
- 点击 "加入游戏" 即可参与 ✨

### 单人游戏（大富翁）
1. 创建房间
2. 点击 ➕ 添加机器人
3. 点击开始，快乐游戏！🤖
