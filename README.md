## 功能特性

### 聊天功能
- 实时群聊消息
- 私聊功能
- 表情发送

- 在线用户列表

## 快速开始

### 环境要求
- Node.js 14.0 或更高版本
- 局域网环境

### 安装步骤

1. 克隆或下载项目
```bash
cd QChat
```

2. 安装依赖
```bash
npm install
```

3. 启动服务器
```bash
node chat-server.js
```

4. 访问应用
   - 本机访问: `http://localhost:3000`
   - 局域网访问: `http://<服务器IP>:3000`

服务器启动后会自动检测本机IP地址并在控制台显示访问地址。

## 使用指南

### 进入聊天室
1. 打开浏览器访问服务器地址
2. 输入你的昵称
3. 如果房间设置了密码，需要输入密码才能进入
4. 点击"进入聊天室"按钮

### 发送消息
- 在底部输入框输入消息
- 点击发送按钮或按 Enter 键发送
- 点击表情按钮可以发送表情

### 私聊
- 点击在线用户列表中的用户名
- 消息将切换为私聊模式
- 再次点击可以切换回群聊

### 主题切换
- 点击右上角的主题按钮切换深色/浅色模式

### 开始游戏
1. 点击聊天室右侧的游戏按钮
2. 选择想要玩的游戏
3. 点击"立即组队"创建游戏房间
4. 其他用户会在聊天室收到游戏邀请
5. 点击"加入游戏"即可参与

## 小游戏介绍

### 五子棋

双人对战策略棋类游戏。

#### 游戏规则
- 黑棋先手，白棋后手
- 双方轮流在棋盘上落子
- 先将五颗棋子连成一线（横、竖、斜）者获胜
- 如果棋盘填满仍无人获胜则为平局

#### 游戏特色
- 15x15 标准棋盘
- 落子动画效果
- 当前回合高亮提示
- 胜利烟花庆祝动画
- 支持观战模式

#### 操作说明
1. 房主创建房间等待对手
2. 对手加入后游戏自动开始
3. 轮到自己的回合时点击棋盘落子
4. 五子连珠即可获胜

### 金钩钓鱼

经典的扑克牌对战游戏，支持 2-4 人游戏。

#### 游戏规则
- 每位玩家初始发 7 张手牌
- 桌面中央翻开一定数量的牌作为"鱼塘"
- 玩家轮流出牌，将手牌放到鱼塘
- 如果出的牌与鱼塘中某张牌点数相同，则"钓走"这两张牌及中间的所有牌
- 钓到的牌归入自己手牌
- 无法钓鱼时只能出一张牌到鱼塘
- 手牌最先出完的玩家获胜

#### 游戏特色
- 实时回合提示
- 牌数统计
- 动画效果
- 游戏时长限制（15分钟）
- 胜利结果同步到聊天

#### 操作说明
1. 点击手牌出牌
2. 如果能钓鱼（有相同点数），自动钓走
3. 策略性地选择出牌顺序

### 翻翻乐

双人记忆配对游戏，考验记忆力。

#### 游戏规则
- 游戏开始时有 16 张卡牌正面朝下
- 每张卡牌都有一个配对的伙伴
- 玩家轮流翻开两张卡牌
- 如果两张卡牌图案相同，配对成功，获得 1 分，可继续翻牌
- 如果两张卡牌不同，自动翻回，换对方翻牌
- 所有卡牌配对完成后，得分高者获胜

#### 游戏特色
- 精美的卡牌图案
- 翻牌动画效果
- 实时得分显示
- 配对成功特效
- 回合切换动画

#### 操作说明
1. 等待轮到自己
2. 点击卡牌翻开
3. 记住卡牌位置
4. 尽可能多地配对成功

### 大富翁

经典多人策略棋盘游戏，支持 2-5 人游戏，支持机器人对战。

#### 游戏规则

**基本流程**
1. 每位玩家初始资金 $10000
2. 轮流掷骰子，按骰子点数前进
3. 到达不同格子触发不同事件
4. 通过买卖地产、收取租金积累财富
5. 其他玩家破产时，剩余玩家获胜

**地图格子类型**
| 格子类型 | 数量 | 说明 |
|---------|------|------|
| 起点 | 1 | 经过获得 $2000 |
| 国家/地产 | 32 | 可购买、建房 |
| 机会 | 4 | 抽取机会卡 |
| 税收 | 4 | 缴纳税款 |
| 监狱 | 1 | 路过安全 |
| 入狱 | 1 | 下一回合跳过 |
| 免费停车 | 1 | 无事件 |

**地产规则**
- 到达无主的地产可选择购买
- 到达自己的地产可花钱建房子
- 建满 3 座房子自动升级为高级公寓
- 到达别人的地产需支付过路费
- 高级公寓过路费最高

**资金不足**
- 资金不足以支付时可出售资产
- 出售价格为原价的 60%
- 无法支付时宣布破产
- 破产玩家进入观战模式

**机会卡类型**
| 类型 | 示例 |
|------|------|
| 获得金钱 | 彩票中奖获得 $1500 |
| 失去金钱 | 违章停车罚款 $400 |
| 前进后退 | 免费搭车前进 3 步 |

#### 游戏特色
- 40 格环形世界地图
- 32 个国家地产
- 20 张机会卡
- 16 种主题棋子
- 棋子跳动动画
- 机会卡滚动抽奖动画
- 过路费/税收飞行动画
- 胜利烟花庆祝
- **支持机器人 AI 对战**

#### 机器人功能
- 单人也可开始游戏
- 添加 1-4 个机器人
- 机器人自动掷骰子
- 机器人自动买地/建房决策
- 机器人自动支付过路费

#### 操作说明
1. 房主创建房间
2. 等待玩家加入或添加机器人
3. 房主点击开始游戏
4. 选择喜欢的棋子
5. 轮到自己时点击"掷骰子"
6. 到达地产后选择是否购买/建房
7. 策略性经营，让对手破产

- **动画**: CSS3 Animation + Canvas
